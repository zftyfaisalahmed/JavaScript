JavaScript Encapsulation

It is process of building the data with the function acting on that data. It allow us to control the data and validate

Point to Remember
  • Use var keyword to make data member private
  • Use setter method to set the data and getter method to get that data.

Encapsulation allow us to handle an object using the following properties.

Read/write -Here, we use setter methods to write the data and getter methods read that data.

Read only -In this case, we use getter method only

Write only -In this case, we use setter method only
